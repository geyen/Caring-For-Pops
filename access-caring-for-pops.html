Server Management

Use these commands to manage the Caring for Pops server:

Start Server

Run the server using the start script:

./run-caring-for-pops.sh

This will start the server on port 3000.

Stop Server

To stop the running server:

pkill -f "node.*app.cjs"

Or press Ctrl+C if the start script is still running.

Note: The server stores all leads in the leads.json file. Make sure this file has proper permissions if you encounter any issues with lead submissions.
